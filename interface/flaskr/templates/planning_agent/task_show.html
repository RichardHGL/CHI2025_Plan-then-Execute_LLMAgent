Action View

Plan Design

  • A clear plan with meaningful instruction can help provide a correct final outcome. To achieve that goal, participants should edit / delete / add / split steps to the plan generated.
  • Please notice that, Each primary step (e.g., 1.) including all sub-steps (e.g., 1.x, 1.x.y) will be used to generate one action to execute in the execution stage. The intermediate steps are providing details of the execution (e.g., bank card, product name).
  • All potential actions can be viewed with the button under task description (button 'Show Potential Actions'). Please double check your plan with all potential actions before moving forward.

How to improve plan

You can improve the generated draft plan with three actions:

  • Edit plan step: edit the text of generated step.
  • Add new step: after you click the 'add one step' button, you should first input the index for a step you want to add.
  • Delete plan step: Participants should delete one primary step when this step can not correspond to any potential action. Please notice that, deleting one primary step will also cause deleting all sub-steps.
  • Split plan step: As each primary step is supposed to be transformed to single action, participants should split one primary step when it includes more than one actions. This action will insert one new primary step and move the follow-up sub-steps in the original step to new primary step.
